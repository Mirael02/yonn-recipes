# Yonn Recipes

Hey there! Welcome to **Yonn Recipes** — a simple, clean recipe website built with good old HTML. The goal here is straightforward: put together a nice little collection of recipes with descriptions, ingredients, and step-by-step instructions.

## What's in the project?

- A homepage that links to each recipe
- Individual recipe pages with full details — description, ingredients list, and cooking steps
- Recipe images in `.webp` format stored in the `img/` directory

## What skills I have demonstrated here?

So far, this project has been a hands-on practice ground for some core web dev fundamentals:

- Using semantic tags like `<h1>`, `<h3>`, `<ul>`, `<ol>`, `<li>`, `<img>`, and `<p>` to build well-organized pages
- Connecting recipe pages back to the homepage and vice versa
- Adding recipe photos with proper `alt` text and sizing
- Keeping things tidy with separate directories for recipes (`recipes/`) and images (`img/`)
- Structuring recipe content with ordered lists for steps and unordered lists for ingredients
